# 🎮 FingerKick Arena – AI-Assisted Mobile Soccer Game 

---

## 🇬🇧 English Version

A lightweight, fast, and fun mobile soccer game built with Unity and enhanced through AI-assisted development.  
I created this project using AI to improve my skills and better understand the logic behind game development.

---

### 🚀 Features
- Smooth and responsive mobile controls  
- Clean pixel-art visuals  
- AI-assisted code structure  
- Modular prefab-based UI system  
- Horizontal & vertical ScrollView skin browsing  
- Optimized performance for mobile devices  

---

### 🧠 Tech Stack
- Unity (2D)  
- C#  
- AI-assisted development workflow  
- Canvas UI / Prefabs / ScrollView  
- Sprite & animation management  

---


### 🎯 Gameplay Overview
- Control your player with smooth movement  
- Unlock skins through the in-game market  
- Browse skins via horizontal/vertical ScrollView  
- Simple and intuitive UI flow  
- Fully optimized for mobile performance  

---

### 🛠️ Installation
1. Open the project using **Unity 2021+**  
2. Import all assets  
3. Run the scene: `Scenes/MainScene`  
4. Enjoy the game  

---

### 📘 Purpose of This Project
This project demonstrates how AI can accelerate game development and help developers understand core mechanics more efficiently.  
Although AI assisted the coding process, the **game logic, architecture, and design decisions were fully developer-driven**.
